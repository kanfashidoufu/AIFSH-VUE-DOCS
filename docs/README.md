> AIFish-Vue

# 入门

## 安装使用

``` bash
# 从gitee上下载源码
git clone https://gitee.com/alsiso/aifish-vue.git

# 安装依赖
npm install

# 启动监听本地服务 localhost:8080
npm run dev

# 编译
npm run build

# 运行单元测试
npm run unit

# 运行e2e测试
npm run e2e

# 运行全部测试
npm test
```
